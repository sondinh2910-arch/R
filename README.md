Thực Hành Rollback / Revert (Hoàn tác)Tình huống: Giả lập commit nhầm file lỗi temp.txt.Cách giải quyết: Thực hiện hành động xóa file trực tiếp trên giao diện Web để GitHub tự động tạo ra một commit mới phủ quyết và dọn dẹp lỗi của commit trước đó mà không làm mất lịch sử dự án.🤖 Tôi Sẽ Dùng Git Như Thế Nào Khi Vibe Code Với AI?Không sửa đổi trực tiếp trên nhánh main: Giữ nhánh main luôn là nơi chứa code sạch và chạy ổn định nhất.Quản lý nhánh nghiêm ngặt: Tạo nhánh phụ (branch) độc lập riêng biệt cho từng tính năng trước khi nhờ AI hỗ trợ sinh code.Kiểm tra và kiểm thử: Sau khi AI sinh code, rà soát lại lỗi, chạy thử nghiệm tại local, thực hiện commit từng bước nhỏ rõ nghĩa.Hợp nhất qua Pull Request: Chỉ đưa code của AI vào dự án chính thông qua luồng Pull Request và kiểm tra xung đột kỹ càng trước khi merge.Cứu hộ thần tốc: Nếu AI sinh code lỗi, sử dụng lịch sử thay đổi để revert / rollback dự án về trạng thái ổn định gần nhất một cách dễ dàng.
